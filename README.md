# Zibzie Health Core

Zibzie Health Core is the shared health-record backend and simple clinical panel for zibzie health products.

This project is focused on building a reusable source of truth for health data.

## Scope

This project includes:

- Patient health profile
- Medical history
- Allergies
- Medications
- Medical documents
- Clinical notes
- Care plans
- Care plan items
- Access control
- Audit logs

## Out of Scope for v1

This project does not include:

- Payment
- VIP registration workflow
- Transportation operations
- Lab booking operations
- Government health gateway integration
- Full DigiCare clinic operations
